# BarnBoss
# Barn Management Game

Конзолна игра за управление на ферма, разработена на C++, в която играчът отглежда култури и животни, събира продукти, изпълнява задачи и развива своята ферма.

## Описание

Проектът симулира фермерско стопанство, в което играчът управлява:

* Земеделски култури
* Животни
* Склад (хамбар)
* Финанси
* Задачи и награди
* Пазар за покупка и продажба на продукти

Целта е да се развива фермата чрез ефективно управление на ресурсите, изпълнение на задачи и разширяване на наличните площи.

## Основни функционалности

### Пазар

Играчът може да:

* Купува семена
* Купува животни
* Продава произведени продукти
* Разглежда наличните продукти на пазара

Налични продукти:

| ID | Продукт            |
| -- | ------------------ |
| 1  | Семена за пшеница  |
| 2  | Семена за царевица |
| 3  | Кокошка            |
| 4  | Крава              |
| 5  | Пшеница            |
| 6  | Царевица           |
| 7  | Яйца               |
| 8  | Мляко              |

### Отглеждане на култури

В играта могат да се засаждат:

* Пшеница
* Царевица

Всяка култура има определен брой цикли на растеж. След достигането им реколтата може да бъде събрана и добавена в хамбара.

### Отглеждане на животни

Играчът може да закупува:

* Кокошки – произвеждат яйца
* Крави – произвеждат мляко

След необходимия брой цикли животните започват да произвеждат съответните ресурси.

### Хамбар

Хамбарът служи за съхранение на всички произведени продукти.

Основни функции:

* Добавяне на продукти
* Премахване на продукти
* Преглед на наличните количества

### Система със задачи

Играчът получава различни задачи за доставка на продукти.

Примери:

* Доставяне на пшеница
* Доставяне на мляко
* Доставяне на яйца

При успешно изпълнение се получават:

* Пари
* Точки

### Разширяване на фермата

Фермата може да бъде разширявана чрез:

* Увеличаване на обработваемата земя
* Увеличаване на капацитета за животни

За разширение са необходими:

* Достатъчно средства
* Определен брой точки

## Икономическа система

### Начални стойности

* Баланс: 100
* Точки: 0

### Начини за печалба

* Продажба на продукти
* Изпълнение на задачи

### Разходи

* Покупка на семена
* Покупка на животни
* Разширяване на фермата

## Структура на проекта

### Основни класове

| Клас            | Предназначение                  |
| --------------- | ------------------------------- |
| `User`          | Базов клас за потребителите     |
| `Player`        | Представя играча                |
| `Farm`          | Управлява културите и животните |
| `Barn`          | Съхранява продуктите            |
| `TaskBoard`     | Списък с активни задачи         |
| `TaskManager`   | Създава и управлява задачите    |
| `MarketManager` | Управлява пазара                |
| `ProductType`   | Дефинира типовете продукти      |
